## Description

A place to see, post, vote and comment on ideas in the style of twitter or 
reddit.


## User stories

- [x] Authenticate users
- [x] Users can CRUD ideas
- [x] Users can upvote and downvote ideas
- [x] Users can bookmark ideas
- [x] Users can comment ideas
  
## Stack

- Server - Nest JS
- Database - PostgreSQL
- REST API - Nest JS
- GraphQL - Apollo
- Auth - JWT
  
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
